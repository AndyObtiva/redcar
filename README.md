Redcar
======

by Daniel Lucraft
http://RedcarEditor.com/

DESCRIPTION
-----------
  
A pure Ruby text editor for Gnome. Has syntax highlighting,
snippets, macros and is highly extensible.

FEATURES
--------
  
* Syntax highlighting for many languages.
* Extensive snippets.
* Ruby plugins.

INSTALLATION
------------

See INSTALL.md

FEATURES
--------

    rake features
    rake features:[plugin_name]
