Object

Certain useful methods are added to Object.

doc The document in the currently focussed tab, if any.
tab The currently focussed tab, if any.
win The currently focussed window.
bus The Redcar Freebase 2.0 Databus

Redcar::App

Redcar::App is a module that controls the application state. There is only ever one instance of the databus per application, and it is accessible through this module. Redcar::App controls the creation of Redcar::Windows, and if ever all Redcar::Windows are closed, the application will terminate.

App.quit Closes all tabs, stops all plugins, and quits the application
App.bus The application databus.
App.new_window (focus = true) Opens a new window.
App.windows Returns an array of open Redcar Windows (or [])
App.focussed_window Returns the currently focussed window.
App.close_all_windows Alias for App.quit

Redcar::Window

A Redcar::Window inherits from Gtk::Window. Windows provide access to their child Panes and a Window will always have at least one child Pane. A Window can (and should) be used to control the creation of Tabs, but in fact the Window delegates this to its child Panes.

close Closes the Window. (Will quit if no more Windows.)
panes Returns an array of the Window's child panes.
unify_all Unifies all child panes.
new_tab (tab_class, focus = true) Opens a new Tab of class tab_class in the appropriate child Pane.
tabs Returns an array of all tabs in this window (or []).
active_tabs Returns an array of the visible tabs in all panes (or []).
focussed_tab Returns the currently focussed tab in this window (or nil).
focus_previous_tab Focusses the previously focussed tab.
focus_tab (name) Focusses the tab titled name.

Redcar::Pane

Redcar::Panes wrap Gtk::Notebooks inside a nested set of Gtk::VBoxs and Gtk::HBoxs. A Pane controls the creation of Redcar::Tabs.

new_tab (tab_class, focus = true) Opens a new Tab of class tab_class in this pane. Returns the Tab.
close_tab (tab) Closes a Tab.
close_all_tabs Closes all Tabs in this pane.
tabs Returns an array of all tabs in this pane.
active_tab Returns the visible tab in this pane.
collect_all (tab_class) Brings all the tabs of class tab_class to this pane.
split_horizontal Divides this Pane into two Panes: left and right. The Tabs will be placed in the right pane.
split_vertical Divides this Pane into two Panes: top and bottom. The Tabs will be placed in the top pane.
unify Undoes the split that created this pane.

Redcar::Tab

A wrapper for a Gtk::Notebook tab. This is an 'abstract' class that contains no actual Gtk widgets.

close Closes the Tab
title The title of the Tab.
title= (name) Sets the title of the Tab.
move_pane (pane) Moves the Tab into pane.
focus Focusses the tab.

Redcar::TextTab < Redcar::Tab

A Tab containing a View and a Document.

document Returns the Document contained in this Tab.
view Returns the View contained in this Tab.
set_file (filename) Sets the filename of the document.
load Loads the file.
save Saves the file.
modified? Returns whether the document has been modified.

Redcar::Document

The Document class contains the text in a Tab and controls most 'text editing' features.

cursor_offset The offset of the cursor.
cursor_offset= (offset) Sets the offset of the cursor.
cursor_to_line_start Moves the cursor to the start of the line.
cursor_to_line_end Moves the cursor to the end of the line.
line_start (n) Returns the offset of the start of line n.
line (n) Returns the contents of line n, including the trailing \n
selection Returns the current selection, or "".
num_lines Number of lines in the buffer.
num_chars Number of chars in the buffer.
selection_offset The offset of the selection point.
selection_bounds [cursor_offset, selection_offset]
forward_character Moves the cursor forward a character.
backward_character Moves the cursor backward a character.
forward_word Moves the cursor forward a word.
backward_word Moves the cursor backward a word.
select_left, select_right Extend the current selection left or right.
select_up, select_down Extend the current selection up or down.
page_up, page_down Move the cursor up or down by a page worth of text.
cut Cut the selection to the clipboard.
copy Copy the selection to the clipboard.
paste Paste the last item on the clipboard over the current selection.
find_next (string|re) Find next occurrence of the string or regular expression argument.
find_previous (string|re) Find previous occurrence of the string or regular expression argument.
insert (offset, text) Inserts the string text at offset.
insert_at_cursor (text) Inserts the string text at the cursor position.
delete (start_offset, end_offset) Deletes the text between start_offset and end_offset.
delete_selection Deletes the selected text, if any.
select (from, to) Selects the text between offsets from and to.
replace_all (text) Replaces the entire contents of the tab with text.
replace_line (n, text) {|line| ...} Replaces line n with text, or the value returned by the block. If n is not specified, then it will replace the current line.
replace_selection (text = nil) {|seln| ...} Replaces the selection with text, or the value returned by the block.
can_undo? Any undo information in this tab.
can_redo? Any redo information in this tab.
undo Undoes previous text changes.
redo Redoes previous text changes.
set_bookmark Sets a bookmark on the current line.
clear_bookmark Removes a bookmark on the current line.
clear_all_bookmarks Clears all bookmarks.
next_bookmark Moves the cursor to the start of the line with the next bookmark.
previous_bookmark Moves the cursor to the start of the line with the previous bookmark.

Redcar::View

scope_at_cursor Returns the scope at the cursor position as a string.

Redcar::Project

objects Returns files and folders in the current project. ["README", "lib/"]
add_object (file_or_dir) Add a file or folder to the project.
remove_object (file_or_dir) Removes the file or folder from the project.
open_file (file) Open a file in a new tab, or focus a tab if already open.
open_folder (folder) Opens a folder in the project tab.
