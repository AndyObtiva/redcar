Roadmap

Redcar 1.0 - Red Editor

This release should have all the features of an excellent text editor, and a wicked project plugin. It should have strong support for extensions/plugins but need not have great IDE features in the core at this point.

  • Editor

    • basic editing commands (Cut, Copy, Paste, Select)
    • syntax highlighting
    • snippets
    • text editing API
    • code folding (even though I hate it)
    • column selection/typing (needed?)
    • auto-completion
    • select scope
    • typing '(' with selection puts (selection)
    • reflow paragraph
    • symbols

  • Commands and Keybindings

    • keymap system
    • multiple commands for a keybinding give a popup menu
    • sensitive to scope
    • command history
    • textmate shell command support
    • macros
    • textmate macro import
    • command editor
    • bundle editor

  • Project Plugin

    • project tab
    • jump to file dialog
    • project shell variables

  • Tabs

    • HtmlTab
    • ShellTab

  • Other Textmate Support

    • document templates
    • shell variable editor
    • tm_dialog?

  • Plugin Support

    • plugin dependencies
    • gem plugins
    • plugin reloading
    • plugin testing
    • databus slot mocking for tests
    • coverage reporting for plugin tests

Redcar 2.0 - Red IDE

This release will be where we can call Redcar an IDE without blushing.

  • Ruby IDE

    • class browser
    • test/unit integration
    • rpsec integration
